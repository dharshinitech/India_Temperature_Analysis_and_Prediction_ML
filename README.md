### 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn (Linear Regression)

### 📊 Project Workflow
* **Data Cleaning:** Handling missing values and formatting date/year columns for analysis.
* **Exploratory Data Analysis (EDA):** Visualizing temperature rise trends over the decades in major Indian cities.
* **Feature Engineering:** Extracting relevant features to train the model.
* **Model Training:** Training a Linear Regression model to find the correlation between time and temperature rise.
* **Future Prediction:** Predicting the average temperature for the year 2030 based on historical patterns.
